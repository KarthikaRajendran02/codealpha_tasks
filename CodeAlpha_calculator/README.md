Build a Calculator

A simple, interactive, and responsive Calculator Web Application built using HTML, CSS, and JavaScript as part of the CodeAlpha Web Development Internship – Task 2.
The calculator supports all basic arithmetic operations with real-time result display and an intuitive user interface.

Project Objective

->To develop a functional calculator that:

->Performs basic arithmetic operations

->Provides a clean and user-friendly interface

->Handles user input efficiently

->Displays results in real time

Features

->> Addition

->> Subtraction

->> Multiplication

->> Division

->> Clear / Reset functionality

->> Display screen for input and output

->> Keyboard input support (Bonus feature)

->> Styled UI with hover effects and transitions

->> Fully responsive design

Technologies Used

~ HTML5 – Calculator structure

~ CSS3 – Styling, layout, and responsiveness

~ JavaScript – Calculator logic and event handling
