Image Gallery Web Application

A responsive and interactive Image Gallery built using HTML, CSS, and JavaScript.
This project showcases a modern gallery layout with smooth navigation, hover effects, and a lightbox view for an enhanced user experience.

Features

->Responsive Design – Works seamlessly on desktop, tablet, and mobile devices

->Image Gallery Layout using HTML & CSS Grid/Flexbox

->Next / Previous Navigation Buttons

->Lightbox View for enlarged image preview

->Hover Effects & Smooth Transitions

->Image Categories / Filters (Bonus Feature)

->Smooth Animations using CSS & JavaScript

Technologies Used

  HTML5 – Structure of the web page

  CSS3 – Styling, animations, transitions, and responsive layout

  JavaScript (ES6) – Navigation, lightbox functionality, and filters
